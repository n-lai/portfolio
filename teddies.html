Teddy Survival

FRONT END ENGINEER

Survival of the Teddy is a game built with JavaScript, Canvas, and HTML/CSS. It features a teddy as the protagonist that has to dodge enemy werewolves.

Visit this live at n-lai.github.io/Teddy-Survival

The game is coded with object-oriented Javascript and utilizes vector arithmetic and a timed Javascript interval to generate dynamic gameplay. Event listeners record the arrow key the user is currently pressing down and determine the direction of the teddy.

Gameflow is commenced and regulated through callback functions and SweetAlert - a JS library that features pre-styled modals. When the game is over, the modal allows users to restart the game and also inform them of their game score.
