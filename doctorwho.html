Doctor Who Tab

FRONT END ENGINEER

Doctor Who Tab is a chrome extension built with React.js, JavaScript, and HTML/CSS. It replaces the standard Google chrome new tab with a Doctor Who themed to-do list and inspiring quote.

Visit this live at the Chrome Web Store

Background images and quotes are stored in arrays. A random number generator determines which wallpaper and quote will be shown.

The to-do list is built with React.js and the tasks are stored in chrome.local storage. A listener is added to chrome storage and updates the to-do list as the user deletes and adds new tasks.
